# Reference Implementation Files

This directory contains the original implementation files kept for reference purposes.

## Files:

1. `modManager.txt` - Original mod management implementation
   - Basic mod downloading
   - Mod listing
   - Mod removal

2. `games.txt` - Original games configuration format
   - Game structure
   - Mod support flags
   - Update URLs

3. `updateChecker.txt` - Original update checking implementation
   - Version comparison
   - Update downloading
   - Basic error handling

These files are kept for:
- Implementation reference
- Migration guidance
- Historical documentation

Note: These are not active files in the project. See the `src/` directory for current implementations.
